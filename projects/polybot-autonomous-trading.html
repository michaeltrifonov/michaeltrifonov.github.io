Hybrid Intelligence Trading Strategy

The GenericStrategy implements a sophisticated multi-stage decision pipeline:

1. Market Discovery & Pre-analysis

Fetches markets via GammaAPI, converts to Pydantic models, and pre-fetches order book data with intelligent caching.

2. Quantitative Gating Pipeline

A suite of specialized analyzers rigorously vet markets before LLM interaction:

PriceAnalyzer
  • Detects arbitrage opportunities (Dutch books)
  • Calculates market efficiency using Shannon entropy
  • Identifies pricing inefficiencies
TimeSeriesAnalyzer
  • Ensemble forecasting: ARIMA, Moving Averages, Linear Regression, EWMA
  • Dynamic ARIMA order selection based on data characteristics
  • Bayesian change point detection
  • Probability distribution shape analysis (skewness, kurtosis)
  • Cross-timeframe probability analysis
LiquidityAnalyzer
  • Order book microstructure analysis
  • Informed trading pattern detection
  • Market impact assessment
  • Effective liquidity calculation
AnomalyDetector
  • Isolation Forest on price, momentum, volatility features
  • Information-adjusted volatility metrics
PositionSizer
  • Multi-factor adjusted Kelly Criterion
  • Dynamic risk adjustment based on market conditions
3. AI-Powered Research & Decision Making

If quantitative analysis passes:

  • Perplexity AI: Conducts time-sensitive web research via api/openai.py
  • Claude 3.7 Sonnet: Synthesizes findings for final trade decision via api/anthropic.py
  • Structured output using tool-calling for reliability

async def execute_order(self, order: Order) -> ExecutionResult:
    # Run blocking py_clob_client in separate thread
    result = await asyncio.to_thread(
        self.clob_client.execute,

# Example artifact persistence
async def persist_artifact(self, artifact: BaseModel):
    """Persist Pydantic model to Cosmos DB with validation"""
    # Validate against shared schema
    validated = self.validate_schema(artifact)
    
    # Generate structured ID
    artifact_id = self.generate_artifact_id(
        type=artifact.__class__.__name__,
        market_id=getattr(artifact, 'market_id', None),
        stage=getattr(artifact, 'stage', 'default')
    )
    
    # Enrich with metadata
    enriched = self.enrich_metadata(validated, artifact_id)
    
    # Save to Cosmos DB with retry logic
    await self.cosmos_client.upsert_item(enriched)
  • Azure Cosmos DB: NoSQL document store for all trading artifacts
  • Schema Validation: Shared Pydantic models via polybots_shared
  • Structured IDs: Queryable artifact identification system
  • Retry Logic: Exponential backoff for rate limiting

@router.get("/artifacts/{artifact_type}")
async def list_artifacts(
    artifact_type: str,
    market_id: Optional[str] = None,
    start_time: Optional[datetime] = None,
    end_time: Optional[datetime] = None,
    limit: int = 100
):
    query = build_cosmos_query(
        artifact_type=artifact_type,
        filters={
            "market_id": market_id,
            "timestamp": {"$gte": start_time, "$lte": end_time}
        }
    )
    
    results = await cosmos_client.query_items(query, limit=limit)
    return {"artifacts": results, "count": len(results)}
